# Intent-Response

## ทักทาย
- **Trigger:** `สวัสดี`
- **Response:** Question: "สวัสดีครับ มีอะไรให้ช่วย?" + buttons (แนะนำตัว, คำสั่ง, ทักทาย)

## แนะนำตัว
- **Trigger:** `แนะนำตัว`
- **Response:** เริ่ม OnboardingConversation → ถามชื่อ → สรุป

## สอนการใช้งาน
- **Trigger:** `สอนการใช้งาน`
- **Response:** เริ่ม OnboardingConversation (เหมือน แนะนำตัว)

## ช่วยเหลือ
- **Trigger:** `ช่วยเหลือ`
- **Response:** "พิมพ์ สวัสดี เพื่อทดสอบ, แนะนำตัว เพื่อเริ่มสอนการใช้งาน, ยกเลิก เพื่อยกเลิก, หรือส่งรูปภาพ/ตำแหน่งที่ตั้ง/สติกเกอร์มาได้"

## รูปภาพ
- **Trigger:** `[Image]`
- **Response:** "รับรูปภาพแล้ว"

## ยกเลิก (มี conversation)
- **Trigger:** `ยกเลิก`
- **Response:** "ยกเลิกการสอนการใช้งานแล้ว" → conversation จบ
- **หมายเหตุ:** inject โดย BaseConversation::ask()

## ยกเลิก (ไม่มี conversation)
- **Trigger:** `ยกเลิก` → fallback
- **Response:** "ไม่เข้าใจครับ พิมพ์ ช่วยเหลือ เพื่อดูว่าผมทำอะไรได้บ้าง"
- **หมายเหตุ:** ไม่มี hears('ยกเลิก') โดยตรง

## Fallback (ข้อความไม่รู้จัก)
- **Trigger:** —
- **Response:** "ไม่เข้าใจครับ พิมพ์ ช่วยเหลือ เพื่อดูว่าผมทำอะไรได้บ้าง"
