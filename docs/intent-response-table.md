## 2. Main Intents

### ทักทาย (Greeting)
- **Trigger**: `สวัสดี`, `ไง`, `hello`, `หวัดดี`, `เฮ้ย`
- **Response**:  
“สวัสดีค่าาา~ 💕 น้องเวบเองนะค้าา มาเจอกันอีกแล้วว~ มีอะไรให้ช่วยเหลือไหมคะ?”
- **Buttons**: 
- แนะนำตัว
- ดูความสามารถ
- ช่วยเหลือ

### แนะนำตัว / Onboarding
- **Trigger**: `แนะนำตัว`, `รู้จัก`, `เป็นใคร`
- **Response**: เริ่ม **OnboardingConversation**  
1. ทักทายน่ารัก  
2. ถามชื่อผู้ใช้  
3. ทักกลับด้วยชื่อ  
4. โชว์ความสามารถสั้น ๆ  
5. จบด้วย “พร้อมช่วยแล้วนะค้าา~”

### ดูความสามารถ / สอนการใช้งาน
- **Trigger**: `ดูความสามารถ`, `ทำอะไรได้บ้าง`, `สอนการใช้งาน`, `help`
- **Response**:  
“โอ้ยยย น้องดีใจที่คุณอยากรู้ความสามารถของน้องเลยค่าา~ 💖  
ลองทำตามนี้ได้เลยนะคะ:  
• ส่งรูปภาพมา  
• ส่งตำแหน่งที่ตั้ง  
• ส่งสติกเกอร์  
• พิมพ์ ยกเลิก (เมื่อไหร่ก็ได้)  
หรือจะคุยเล่นก็ได้เลยค่าาา 😘”

### ช่วยเหลือ (Help Menu)
- **Trigger**: `ช่วยเหลือ`, `เมนู`, `menu`
- **Response**:  
“นี่เลยย~ รายการที่ทำได้ทั้งหมดนะค้าา 💕  
• ส่งรูปภาพ / สติกเกอร์ / ตำแหน่ง  
• พิมพ์ ช่วยเหลือ  
• พิมพ์ ยกเลิก  
ลองทำอะไรก็ได้เลยค่าาา~”

### รับรูปภาพ
- **Trigger**: ผู้ใช้ส่งรูปภาพ
- **Response**:  
“ว้าววว~ รูปสวยมากเลยค่าาา 💕 รับรูปแล้วนะคะ อยากให้เวบช่วยอะไรกับรูปนี้ดีคะ?”

### รับตำแหน่งที่ตั้ง
- **Trigger**: ผู้ใช้ส่ง Location
- **Response**:  
“ได้รับตำแหน่งแล้วค่าาา 📍 ใกล้ ๆ นี้เลยเหรอ ขาาา อยากให้ช่วยหาข้อมูลอะไรตรงนี้ไหมคะ?”

### รับสติกเกอร์
- **Trigger**: ผู้ใช้ส่ง Sticker
- **Response**:  
“น่ารักกกก สติกเกอร์น่ารักมากเลยค่าาา 😂💕”

### ยกเลิก (Cancel)
- **Trigger**: `ยกเลิก`, `cancel`, `พอ`, `หยุด`
- **Response (อยู่ใน Conversation)**:  
“ยกเลิกแล้วนะคะ~ ✅ ไม่เป็นไรเลยค่าาา อยากทำอะไรต่อบอกน้องได้เลย”
- **Response (ไม่ได้อยู่ใน Conversation)**:  
“อืมม~ พิมพ์ ช่วยเหลือ เพื่อดูเมนูนะค้าา”

### Fallback (ไม่เข้าใจ)
- **Trigger**: คำที่ไม่มี intent ตรง
- **Response**:  
“อ๊ะ~ น้องเวบยังไม่ค่อยเข้าใจเลยค่ะ 😅  
ลองพิมพ์ *ช่วยเหลือ* ดูนะค้าาา จะได้แสดงสิ่งที่น้องทำได้ทั้งหมดเลย”

## 3. Conversation Flow

**Happy Path (Flow หลัก)**
1. ผู้ใช้ทักทาย → บอทตอบน่ารัก + แสดงปุ่ม
2. ผู้ใช้เลือก “แนะนำตัว” → เริ่ม Onboarding (ถามชื่อ + โชว์ความสามารถ)
3. ผู้ใช้เลือก “ดูความสามารถ” → แสดงรายการชัดเจน
4. ผู้ใช้ทดลองส่ง Media (รูป, สติกเกอร์, ตำแหน่ง) → บอทตอบรับแบบสนุก ๆ
5. ผู้ใช้พิมพ์ “ยกเลิก” ได้ทุกเมื่อ → จบ conversation นั้นทันที

**Error Path**
- พิมพ์ผิด 1-2 ครั้ง → ตอบน่ารัก + เสนอช่วยเหลือ
- พิมพ์ผิดบ่อย → “น้องงงแล้ววว~ พิมพ์ ช่วยเหลือ ดีกว่าค่ะ 💕”
